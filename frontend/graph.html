This is Cognee.

Everything you just did — every clue, character, location and code — was stored as a knowledge graph in Cognee Cloud. Cognee is an AI memory layer: it ingests raw text, builds a hybrid vector + graph memory, grounds entities into an ontology, and lets AI agents recall connected facts across sessions.

That's how the four AI minds reconstructed a night nobody remembered — and how they debunked the false leads. The graph behind this card is the real memory Cognee built for this case (not a mockup). Use the dataset picker to explore other memories, like Mr. Chow's personality graph.
